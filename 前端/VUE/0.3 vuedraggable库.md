参考[vuedraggable中文文档](http://www.itxst.com/vue-draggable/tutorial.html)。

### 1. npm安装

```shell
$ npm i -S vuedraggable
```

### 2. 使用

```vue
<template>
	<div>
    <draggable></draggable>
  </div>
</template>

<script>
  // import
import draggable from "vuedraggable";
export default {
  // 注册component
  components: {
    draggable
  }
}
</script>
```
